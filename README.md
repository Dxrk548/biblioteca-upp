# Biblioteca App (monorepo)

Proyecto demo con frontend React (Vite) y backend Node (Express).

Carpetas:
- `frontend` - aplicación React (Vite).
- `backend` - API simple con Express.

Instrucciones rápidas:

1. Abrir terminal en `frontend` y ejecutar:

```powershell
npm install
npm run dev
```

2. En otra terminal, abrir `backend` y ejecutar:

```powershell
npm install
npm start
```

Frontend por defecto corre en `http://localhost:5173` y backend en `http://localhost:4000`.

Credenciales demo (frontend -> backend):
- usuario: `admin`
- contraseña: `password`
